# Nbt

A simple NBT library with support for custom tag types.

## Usage

### Dependency

You can get it via [JitPack](https://jitpack.io/):

```groovy
repositories {
    maven {
        name = "JitPack"
        url = "https://jitpack.io/"
    }
}

dependencies {
    implementation "com.github.AntiquityMC:Nbt:1.0.0"
}
```
